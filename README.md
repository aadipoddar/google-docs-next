Clone it and Run it:

Create a .env.local file and give the following fiels
```
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_URL=
```
Then :-
```bash
yarn
yarn dev
```
